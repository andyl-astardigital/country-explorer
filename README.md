# Country Explorer

See src/doc folder for a list of Architectural Decisions Records (ADRs) for this project.

package.json has a number of react scripts to build, run and test the application.

To get started:
 - npm i
 - npm run start

To unit test:
 - npm run test

To integration test:
 - npm run start
 - npm run e2e


# Tech used
- React
- Typescript
- Material UI
- Axios
- ReactQuery
- React Testing Library
- Cypress
